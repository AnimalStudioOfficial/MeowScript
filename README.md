# MeowScript
A small interpreted easy extendable programming language.  
License: MIT

## Requirements
The requirements are:
- CMake and Make
- a  C++ compiler
- Either Windows or Linux (Other systems not tested, but might work to some extend)

## Installing
Execute this commands in your shell:
```
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./meow-script --help
```
